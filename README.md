# Projeto-Arduino
Projeto voltado ao uso do arduino com outros aplicativos e tecnologias para a liberação de passagem via um cartão ou tag.

## Ferramentas
Foi utilizado:  
  * Arduino
    **RFID e LCD/Led.
  * Node.js
  * MongoDB

## Instalação
Baixar os arquivos, dentro das pastas "Server" e "Client" rodar o "npm install".
